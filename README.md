### `STEPS`

git clone https://github.com/mdali602/next-with-redux.git <br />
cd next-with-redux <br />

To run the app in the development mode.<br>
$npm i <br />
$npm run dev <br />

****  OR  ****

$yarn install <br />
$yarn dev <br />

<br />

To run production build: <br />

$npm run build <br />
$npm start <br />

****  OR  ****

$yarn build <br />
$yarn start <br />
